## 📞 技术支持

### 常见问题

**Q: 系统启动失败?**  
A: 运行 `bash HEALTH_CHECK.sh` 诊断问题，参见 [QUICK_START.md](./QUICK_START.md) 故障排查章节

**Q: API 返回 401 Unauthorized?**  
A: 检查 Authorization header 是否为 `Bearer <YOUR_INTERNAL_API_TOKEN>`

**Q: 小程序无法连接后端?**  
A: 确保:
  - Docker 容器运行中 (`docker ps`)
  - 后端服务使用 (`lsof -i :8000`)
  - 网络配置正确 (`config/finance.config.js`)

**Q: 数据库连接超时?**  
A: 等待数据库就绪 (通常 10-15 秒)，见 `START_SYSTEM.sh`
