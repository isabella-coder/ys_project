# 🛒 云服务器购买指南

## 推荐方案

### 选项 1: 阿里云 (推荐国内用户 - 支持备案)
**链接**: https://www.aliyun.com/

配置:
- 实例规格: t6-burstable(t6) 2核4G
- 镜像: Ubuntu 20.04 LTS
- 带宽: 按使用流量计费 或 固定 5Mbps
- 估算: ¥60-100/月

购买步骤:
1. 登录阿里云账户
2. 产品 → ECS 云服务器
3. 立即购买 → 选择 "按量付费" 或 "包年包月"
4. 选择地域 (最好选靠近用户的地方)
5. 选择镜像: Public Image → Ubuntu 20.04
6. 选择配置: 2vCPU 4GB 内存
7. 完成购买

购买后:
- 记录 **公网 IP 地址** (如 47.98.123.45)
- 记录 **SSH 登录用户名** (通常是 ubuntu)
- 设置 **密码** 或 **密钥对**

---

### 选项 2: 腾讯云 (推荐国内用户 - 支持备案)
**链接**: https://cloud.tencent.com/

配置:
- 机型: 标准型 CVM 2核4G
- 镜像: TencentOS Server 3.1 (或 Ubuntu 20.04)
- 带宽: 按带宽计费 5Mbps
- 估算: ¥70-120/月

购买步骤:
1. 登录腾讯云
2. 产品 → CVM 云服务器
3. 自定义配置
4. 选择地域和可用区
5. 选择镜像: Ubuntu 20.04
6. 配置: 2核 4GB内存
7. 完成购买

---

### 选项 3: DigitalOcean (推荐国际用户)
**链接**: https://www.digitalocean.com/

配置:
- Droplet: Standard Shared CPU
- 规格: 2vCPU 4GB
- 操作系统: Ubuntu 20.04
- 估算: $24-30/月 (约¥150-200)

购买步骤:
1. 登录 DigitalOcean
2. Create → Droplets
3. 选择镜像: Ubuntu 20.04
4. 选择规格: 4GB / 2vCPU
5. 选择数据中心 (推荐新加坡或日本用于中国)
6. 完成创建

---

## 建议

**如果您在国内使用，强烈推荐阿里云或腾讯云**，因为:
1. 需要 **ICP 备案** (国内法律要求)
2. 国内访问速度快 (CDN 已由云厂商优化)
3. 后续金融支付接入更方便

**如果需要国际访问，可选 DigitalOcean**:
1. 全球 CDN
2. 按量付费，多备份选择
3. 但需完成国际备案 (如果有国内用户)
