标准流程

  1. 微信开发者工具点击“上传”，版本号用新号（如 1.0.5）。
  2. 登录 mp.weixin.qq.com，在“版本管理”里选择开发版本并“提交审核”。
  3. 审核通过后执行“发布”（建议分阶段发布）。
  4. 发布后真机抽检关键路径并记录结果。
